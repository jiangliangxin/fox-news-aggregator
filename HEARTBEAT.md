每次心跳时检查是否需要执行任务。

## 定时任务检查

### 每日新闻推送
- 早间推送时间：北京时间 8:00 (UTC 0:00)
- 晚间推送时间：北京时间 20:00 (UTC 12:00)
- 触发消息：包含 "【新闻推送】"
- 工作流程：
  1. 运行脚本：python3 /root/.openclaw/workspace/generate_daily_news.py
  2. Fox阅读日报内容，用自己的理解总结后发送给主人
- 当前版本：v4.0（2026-03-04）
  - ✅ 激活分析引擎（correlation/alerts/entities）
  - ✅ 智能摘要头部（发酵话题、跨源热点、告警、人物）
  - ✅ 历史对比功能
  - ✅ 修复多个bug
- 推送内容（78个数据源，465+条新闻）：
  - 每天60秒读懂世界（15条 + 每日微语）
  - 微博/知乎/B站热搜（各10-30条）
  - IT之家科技资讯（10条）
  - GitHub Trending（5条）
  - Hacker News（5条）
  - Epic免费游戏
  - 国际科技新闻（30条）
  - 财经新闻（国内/国际分开）
  - 统计信息 + 健康状态
- 性能：首次50-60秒，带缓存15-20秒
- 数据源：12个API源 + 27个RSS源 = 39个活跃数据源
- 架构：v4.0插件化架构（sources/ + core/ + analysis/）

### 服务器监控
- 监控频率：每小时 30 分 (UTC)
- 触发消息：包含 "【服务器监控】"
- 执行脚本：python3 /root/.openclaw/workspace/smart_monitor.py（智能告警版本）
- 监控内容：
  - 关键服务：x-ui、openclaw-gateway、dockerd、node-exporter
  - 代理服务：x-ui、xray、错误日志
  - 系统资源：CPU、内存、磁盘
  - 网络连通性：DNS、HTTP
- 告警规则：
  - CPU负载 >= 2.0 (警告) / >= 4.0 (严重)
  - 内存使用 >= 70% (警告) / >= 90% (严重)
  - 磁盘使用 >= 75% (警告) / >= 90% (严重)
- 异常通知：发现问题时立即通知主人
